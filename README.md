# Simple CRUD in using [Nim](https://nim-lang.org/), [Jester](https://github.com/dom96/jester/), and [Norm](https://github.com/moigagoo/norm)
## Running Locally
1. [Install Nim](https://nim-lang.org/) and clone repo.  

2. Install dependencies locally  
```bash
nimble install -l -y
```

3. Build and Run  
```bash
nimble run
```
